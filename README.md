## Technology stack：
+ Vue 2.6.14
+ Vuex 3.6.2
+ Vant 2.12.48
+ Vue-awesome-swiper 4.1.1
## Features
+ Login / Register
+ Obtain product list using local data
+ Add/Remove item to shopping cart
## Get Started
```
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run serve
```
